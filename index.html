Projets en Vedette

Projet personnel : Application Bancaire PHP-MYSQL

Rôle : Développeur full stack

Application web de gestion de compte bancaire. Le projet est hébergé sur alwaysdata et le code source est disponible sur mon compte GitHub. Il inclut comme fonctionnalités : la création et la gestion de compte, un système d'authentification, de virement, de dépôt d'argent, une messagerie instantanée, l'ajout de bénéficiaire, etc.
Technologies : HTML 5, Tailwind Css, Jquery, PHP, Mysql,Git.

Projet personnel : API Rest Node.js

Rôle : Développeur backend

API REST JavaScript, cette application utilise les requêtes HTTP pour effectuer des actions sur une application tierce, telles que la lecture, la modification, la suppression ainsi que la création de données à partir des données fournies depuis un serveur Apache. L'utilisation du framework Node.js avec Express.js est mise en valeur pour faire du JavaScript en backend.
Technologies : Javascript, Node.js, Express.js, Mysql, Git.

Clone du réseau social LinkedIn avec la bibliothèque ou le framework JavaScript React. Le projet comprend un système de routage des différentes pages et plusieurs composants pour les différentes parties de l'application : header, popup, etc.
Technologies : HTML 5, Tailwind css, Javascript, React, Git.
